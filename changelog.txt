Lanai Core is a mature OOP framework for building web, command line and GUI applications. 
Licensed under GNU/GPL, the project allows PHP developers to easily integrate and manage 
code resources, and build complex applications quickly. Support PHP 4/5 compatible, compact 
core framework with basic CMS and other modules included to get you started, ease of use: 
learning curve not as steep as other frameworks enterprise ready, performance conscious 
design tight integration with PEAR libraries, great for prototyping complex projects.
